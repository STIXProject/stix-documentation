AvailabilityLossTypeVocab-1.0STIX Vocabularies Schema

The AvailabilityLossTypeVocab is the default STIX vocabulary for expressing the type of availability that was lost due to an incident. NOTE: As of STIX Version 1.1.1, this version of the AvailabilityLossTypeVocab is deprecated. Please use AvailabilityLossTypeVocab-1.1.1 instead.

Vocabulary Items

Item Description
Destruction The information was destroyed or wiped.
Loss Availability to the information was lost.
Interruption Availability to the information was interrupted.
Degredation Availability to the information was degraded.
Acceleration Availability loss type is acceleration.
Obscuration Availability to the information is obscured.
Unknown The availability loss type is not known.

Fields

Field Name Type Description
@vocab_nameoptional string

The vocab_name field specifies the name of the controlled vocabulary.
@vocab_referenceoptional anyURI

The vocab_reference field specifies the URI to the location of where the controlled vocabulary is defined, e.g., in an externally located XML schema file.
